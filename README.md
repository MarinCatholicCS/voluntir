# Voluntir

A volunteer coordination platform that connects volunteers with community service opportunities. Browse events, sign up, track hours, and climb the leaderboard.

**Live:** [marincatholiccs.github.io/voluntir](https://marincatholiccs.github.io/voluntir/)

## Features

- **Browse Events** — Discover volunteer opportunities with search and filtering
- **Sign Up** — Register for events with a single click
- **Create Listings** — Post your own volunteer events for the community
- **Leaderboard** — Track volunteer hours and compete with individuals and schools
- **Profiles** — Customize your profile with school affiliation, location, and photo
- **Upcoming Events** — View all your registered events in one place

## Tech Stack

- **Frontend:** React + Vite
- **Auth:** Firebase Authentication (Google sign-in)
- **Database:** Cloud Firestore
- **Hosting:** GitHub Pages 

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Deployment

Pushes to `main` automatically build and deploy to GitHub Pages via GitHub Actions. No manual build step needed.
